A Quality of Life mod for Warlords 2
- Respawn timer
- Last Stand detection and timer
- Intervene detection and timer

Download: https://www.mediafire.com/file/aau55ixq85dwylq/warlords2qol-1.0.0.jar/file

Commands:
  Respawn Timer -
    - /timerstop
    - /timerstart
    - /timerset xx:xx y
      + y = set tick (0-20)
    - /timerscale x.x
    - /tiemrgettick
  Cooldown Timers -
    - /cdscale
